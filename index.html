Text Editor Source XML 1.1.0

XML source code editor extension for text editor.

Provides a set of key strokes to generate responses like in a typical XML source code editor.

Previous tabbable element.

Next tabbable element.

Features

  • <div| or <div|> then press >.
  • <div| or <div|> then press /.
  • <div>| then press ↵.
  • <div>|</div> then press ↵.
  • <div>| then press ⌫.
  • <div/>| then press ⌫.
  • <div>|</div> then press ⌫.
  • |<div> then press ⌦.
  • |<div/> then press ⌦.
  • |</div> then press ⌦.
  • <div>| then press ←.
  • <div/>| then press ←.
  • </div>| then press ←.
  • |<div> then press →.
  • |<div/> then press →.
  • |</div> then press →.
  • <| or <|> then press ?.
  • <?| or <?|?> then press ⌫.
  • <!-| or <!-|> then press -.
  • <!-- | --> then press ↵.
  • <!-- | or <!-- | --> then press ⌫.
  • |--> then press ⌦.
  • |?> then press ⌦.

<script>
Object.assign(TE.prototype, TE.History.that);
const editor = new TE(document.querySelector('textarea')),
const {canKeyDown, canKeyDownHistory, canKeyDownTab, canKeyUp} = TE.Source;
const {canKeyDown: canKeyDownXML} = TE.SourceXML;
editor.self.addEventListener('keydown', e => {
    let key = e.key,
        keys = {a: e.altKey, c: e.ctrlKey, s: e.shiftKey};
    if (
        canKeyDownXML(key, keys, editor) &&
        canKeyDown(key, keys, editor) &&
        canKeyDownHistory(key, keys, editor) &&
        canKeyDownTab(key, keys, editor)
    ) {
        // Normal key strokes. Add more key strokes extension here!
    } else {
        e.preventDefault();
    }
});
editor.self.addEventListener('keyup', e => {
    let key = e.key,
        keys = {a: e.altKey, c: e.ctrlKey, s: e.shiftKey};
    canKeyUp(key, keys, editor) || e.preventDefault();
});
editor.record(); // Record the first history data
</script>

Strokes

Strokes can either return false or true. If it returns false, it means that this extension has manipulated the existing value. If it returns true, it means that current stroke works normally.

TE.SourceXML.canKeyDown(key, {a, c, s}, that)

This will handle tasks such as automatic XML tag closing and automatic indentation by pressing the ↵ key.
